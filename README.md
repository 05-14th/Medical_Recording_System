# Medical Recording System

A comprehensive PHP-based web application designed for managing and recording medical information securely.

## 📋 Overview

The Medical Recording System is a full-featured application built with PHP, CSS, and HTML to streamline medical record management, patient information tracking, and healthcare data organization. This system provides a user-friendly interface for healthcare professionals to efficiently manage patient records and medical documentation.

## 🛠️ Technology Stack

- **Backend:** PHP (90.9%)
- **Frontend:** HTML (0.8%) & CSS (8.3%)
- **Database:** Compatible with standard relational databases

## ✨ Features

- **Patient Record Management** - Create, update, and manage patient medical records
- **Secure Data Storage** - Healthcare data stored with security best practices
- **User-Friendly Interface** - Intuitive web-based dashboard
- **Medical Information Tracking** - Organize and retrieve patient information efficiently

## 🚀 Getting Started

### Prerequisites

- PHP 7.0 or higher
- Web server (Apache, Nginx, etc.)
- MySQL or compatible database
- Modern web browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/05-14th/Medical_Recording_System.git
cd Medical_Recording_System
```

2. Configure your database connection in the application configuration file

3. Set up proper file permissions for the application directories

4. Access the application through your web server

## 📁 Project Structure

```
Medical_Recording_System/
├── index.php              # Main entry point
├── assets/                # CSS, images, and static files
├── includes/              # Reusable PHP components
├── pages/                 # Application pages
└── database/              # Database configuration and queries
```

## 💻 Usage

1. Access the application through your web browser
2. Log in with your credentials
3. Navigate to the patient records section
4. Create, view, or edit medical records as needed
